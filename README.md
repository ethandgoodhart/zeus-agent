## 🔥 features

- 🧠 Intelligent task planning and execution
- ⚡ Lightning-fast response time
- 🖥️ Native macOS integration via Accessibility APIs
- 🔄 Real-time DOM analysis of applications

## 🚀 Getting Started
### Install

```bash
pip install -r requirements.txt
```

### Running

```bash
python agent.py
```

### Prerequisites

- macOS (10.15+)
- Python 3.8+
- Gemini API key
